## Configuration

Here's what you need to add to your tauri.conf.json file:
```json
"tauri": {
  "embeddedServer": {
    "active": false               // do not use a localhost server
  },
  "allowlist": {                  // all API endpoints are default false
    "event": true,                // Use the EVENT API for injections
  }
}

```
