:::tip `create-tauri-app`

The easiest way to scaffold a new project is the [`create-tauri-app`](https://github.com/tauri-apps/create-tauri-app) utility. It has support for vanilla HTML/CSS/JavaScript but also many frontend frameworks like React, Vue, and Svelte.

```

sh <(wget https://create.tauri.app/sh)
```

</TabItem>
