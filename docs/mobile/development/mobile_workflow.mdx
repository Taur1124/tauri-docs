## Generate Mobile setup

<Tabs groupId="platform">
  <TabItem value="android" label="Android">

After you installed the [Android dependencies](#setup-android-dependencies) use the Tauri CLI to generate the required Android source code:

<Command name="android init" />

  </TabItem>

  <TabItem value="ios" label="iOS">

After you installed the [iOS dependencies](#setup-ios-dependencies) use the Tauri CLI to generate the required iOS source code:

<Command name="ios init" />

  </TabItem>
</Tabs>

## Development cycle

<Tabs groupId="platform">
  <TabItem value="android" label="Android">

Now you can use the `dev` command to run the app in an existing emulator or a connected device directly from the command line.
You can also use the `--open` flag to use Android Studio instead.

<Command name="android dev [--open]" />

To build the application bundles (APK and AAB files), run the `build` command.

:::caution
On release builds it is required to sign your application.
:::

<Command name="android build" />

  </TabItem>

  <TabItem value="ios" label="iOS">

Now you can use the `dev` command to run the app in a simulator or a connected device directly from the command line.
You can also use the `--open` flag to use Xcode instead.

<Command name="ios dev [--open]" />

To build the application bundles (IPA file), run the `build` command.

<Command name="ios build" />

  </TabItem>
</Tabs>
