## Generate Mobile setup

<Tabs groupId="platform">
  <TabItem value="android" label="Android">

After you installed the [Android dependencies](#setup-android-dependecies) use the `tauri-cli` to generate the required Android source code:

<Command name="android init" />

  </TabItem>

  <TabItem value="ios" label="iOS">

After you installed the [iOS dependencies](#setup-ios-dependencies) use the `tauri-cli` to generate the required iOS source code:

<Command name="ios init" />

  </TabItem>
</Tabs>

## Development cycle

<Tabs groupId="platform">
  <TabItem value="android" label="Android">

Now you can either use the `open` subcommand to open the generated project in Android Studio or use the `dev` and `build`
subcommands if you haven't installed Android Studio.

<Command name="android open" />

<Command name="android dev" />

  </TabItem>

  <TabItem value="ios" label="iOS">

Now you can either use the `open` subcommand to open the generated project in Xcode or use the `dev` and `build`
subcommands to run it from the commandline.

<Command name="ios open" />

<Command name="ios dev" />

  </TabItem>
</Tabs>
