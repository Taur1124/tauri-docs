Make HTTP requests with the http plugin.

## Setup

Install the http plugin to get started.

<Tabs>
    <TabItem label="Automatic">

    1. Use your project's package manager to add the dependency:

    <CommandTabs npm="npm run tauri add http"
    yarn="yarn run tauri add http"
    pnpm="pnpm tauri add http"
    cargo="cargo tauri add http" />

    2. Modify `lib.rs` to initialize the plugin:

    {/* TODO: Revise once https://github.com/tauri-apps/tauri/issues/7696 is in */}

    ```rust
    #[cfg_attr(mobile, tauri::mobile_entry_point)]
    pub fn run() {
        tauri::Builder::default()
            // Initialize the plugin
            .plugin(tauri_plugin_http::init())
            .run(tauri::generate_context!())
            .expect("error while running tauri application");
    }
    ```

    </TabItem>
    <TabItem label="Manual">

    1. Run `cargo add tauri-plugin-http` to add the plugin to the project's dependencies in `Cargo.toml`.

    2. Modify `lib.rs` to initialize the plugin:

    ```rust
    // lib.rs
    #[cfg_attr(mobile, tauri::mobile_entry_point)]
    pub fn run() {
        tauri::Builder::default()
            // Initialize the plugin
            .plugin(tauri_plugin_http::init())
            .run(tauri::generate_context!())
            .expect("error while running tauri application");
    }
    ```

    3. If you'd like to make http requests in JavaScript then install the npm package as well:

    <CommandTabs
        npm="npm install @tauri-apps/plugin-http"
        yarn="yarn add @tauri-apps/plugin-http"
        pnpm="pnpm add @tauri-apps/plugin-http"
    />

    </TabItem>

## Usage

The http plugin is available in both JavaScript and in Rust, as a [reqwest](https://docs.rs/reqwest/) re-export.

#### 1. Configure allowed scope

```json
//tauri.conf.json
{
"plugins": {
	"http": {
	"scope": ["http://my.api.host/*"]
		}
	}
}
```
#### 2. Send a request
<Tabs>
<TabItem label="JavaScript">

```js
import { fetch } from '@tauri-apps/plugin-http';

// Send a GET request
const res = await fetch('http://my.api.host/data.json');

console.log(res.status); // e.g. 200
console.log(res.statusText); // e.g. "OK"

// Send a POST request
const res = await fetch("http://my.api.host/post", {
	method: "post",
	body: JSON.stringify({
	title: "foo",
	body: "bar",
	userId: 1,
	}),
	headers: {
	"Content-type": "application/json; charset=UTF-8",
	},
});
console.log(res.status); // e.g. 201
console.log(res.statusText); // e.g. "Created"
```

Note: The `fetch` method is not the same as `fetch` from Web API. It instead utilizes a Rust package to send the requests.

</TabItem>
<TabItem label="Rust">

In Rust you can utilize the reqwest crate via `use tauri_plugin_http::reqwest;`. For more details refer to [reqwest docs](https://docs.rs/reqwest/).


```rust
use tauri_plugin_http::reqwest;

let res = reqwest::get("http://my.api.host/data.json").await;
println!("{:?}", res); // print response to terminal



```
